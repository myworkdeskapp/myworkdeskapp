Super Admin Credentials

Super admin credentials (username, security key, and password) are managed via platform environment variables: SA_USERNAME, SA_SECURITY_KEY, and SA_PASSWORD. To rotate credentials, update these variables in your Cloudflare Pages project settings and redeploy.
Cloudflare Secret Variables
SA_USERNAME
—
SA_SECURITY_KEY
—
SA_PASSWORD
—
Run these commands to set the missing secrets
wrangler secret put SA_USERNAME     --name workdesk-worker
wrangler secret put SA_SECURITY_KEY --name workdesk-worker
wrangler secret put SA_PASSWORD     --name workdesk-worker
Enter each value when prompted, then redeploy the project.
